Adding your domain name:

This should be the first thing you do.

  • Login to your control panel
  • Click 'Domains'
  • This page allows you to add new domain names
  • Then go to Advanced --> DNS Settings --> Select your domain name --> Click 'add default records'

The last thing to do is change the name server settings with your domain registrar to the following:

$ns1
$ns2

Domain names should resolve to our service within 1 hour, however can take up to 48 hours.

FTP:

  • Login to your control panel
  • Click 'FTP Accounts'
  • Create a username and password

FTP can then be accessed using a range of free programs (we recommend FileZilla) with the address line:

ftp.[your-domain-name].[ext]
