Creating a payment method:

Insert HTML to create a payment method. This HTML will be displayed on the payment options page. Where values such as the invoice ID, user alias, payment amount should be entered, enter the following variables instead. For example, "<input type="hidden" name="a1" value="$itmvalue" />" $itmvalue will be replaced with the fee.

Where using the PayPal API for automatic enabling of accounts, read the following carefully.

Variables:

$itmvalue = amount the user will pay e.g. 5
$userid = the user alias entered during signup. For example 'zpanel'.
$invid = the unique ID for the invoice created during signup
$subscrper = used for subscription based payment links. Will set the payment interval. For example, if the user selected monthly subscription, this will be '1', if user selected quarterly payment intervals this will be '3' and for yearly payment intervals this will be '12'.

PayPal API:

For this to work, set item_number to '$userid'

Add this to the button HTML to ensure accounts are activated in real-time:
"<input type="hidden" name="notify_url" value="[WEB-URL-TO-SCRIPT]/PayPalAPI.php" />"
 

PayPal Subscription based API:

For this to work, set item_number to '$invid'
"<input type="hidden" name="notify_url" value="[WEB-URL-TO-SCRIPT]/PayPalAPI-Subsc.php" />"

Sandbox

For testing purposes, use the PayPal Sandbox.

Edit the PayPalAPI.php and PayPalAPI-Subsc.php file:
- replace https://www.paypal.com with https://sandbox.paypal.com

Ensure your payment links point to sandbox.paypal.com also.

Sample PayPal Subscription data:

This will setup a recurring subscription, all you need to do is add your PayPal Email in the bit in red.

<form action="https://www.paypal.com/cgi-bin/webscr" method="post">
<input type="hidden" name="cmd" value="_xclick-subscriptions" />
<input type="hidden" name="no_note" value="1" />
<input type="hidden" name="a3" value="$itmvalue" />
<input type="hidden" name="p3" value="$subscrper" />
<input type="hidden" name="t3" value="M" />
<input type="hidden" name="business" value="[YOUR-PAYPAL-EMAIL]" />
<input type="hidden" name="src" value="1" />
<input type="hidden" name="srt" value="0" />
<input type="hidden" name="sra" value="1" />
<input type="hidden" name="modify" value="0" />
<input type="hidden" name="usr_manage" value="0" />
<input type="hidden" name="item_name" value="ACC: $userid" />
<input type="hidden" name="currency_code" value="GBP" />
<input type="hidden" name="lc" value="GB" />
<input type="hidden" name="charset" value="windows-1252" />
<input type="hidden" name="item_number" value="$userid" />
<input type="hidden" name="notify_url" value="http://billing.interwebproject.com/test/PayPalAPI-Subsc.php" />
<input type="image" src="https://www.paypal.com/en_GB/i/btn/btn_subscribeCC_LG.gif" border="0" name="submit" alt="PayPal - The safer, easier way to pay online!" />
</form>
